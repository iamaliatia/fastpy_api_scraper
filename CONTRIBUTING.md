# Contributing to FastPy-API-Scraper

First off, thank you for considering contributing to FastPy-API-Scraper! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## 📜 Code of Conduct
By participating in this project, you agree to maintain a respectful, welcoming, and professional environment for everyone.

## 🚀 How Can I Contribute?

### 🐛 Reporting Bugs
- Check the [Issues tab](https://github.com/iamaliatia/fastpy_api_scraper/issues) to ensure the bug hasn't already been reported.
- If it's a new bug, open a new Issue and include:
  - A clear summary of the problem.
  - Steps to reproduce the behavior.
  - Your environment details (OS, Python version).

### ✨ Suggesting Enhancements
- Open an Issue with the tag `enhancement`.
- Describe the feature you want to see added and why it's useful.

### 🛠️ Submitting Code (Pull Requests)
If you want to fix a bug or add a feature from our Roadmap:
1. **Fork** the repository to your own GitHub account.
2. **Clone** your fork locally:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/fastpy_api_scraper.git](https://github.com/YOUR_USERNAME/fastpy_api_scraper.git)
